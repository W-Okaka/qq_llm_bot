# QQ LLM Bot

基于 NoneBot2 + OneBot V11 的 QQ 聊天机器人，支持多模型切换与群聊技能路由（聊天、画图、搜索、语音等）。

## 开发说明

本项目代码由 Claude Code 协助开发（纯 Claude Code 开发流程）。

## 仓库可见性说明

本仓库按 `Public` 方式发布，任何人都可以查看源码与提交记录。

## 隐私与安全

以下内容不会上传到仓库：

- `.env`（API Key 等密钥）
- `src/plugins/llm_chat/data/`（聊天历史、记忆、运行时数据）
- `CLAUDE.md`（本地隐私与个性化指令）
